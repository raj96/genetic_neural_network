Parameters
populationPointer to a ga_population struct to run the algorithm on
ngenNumber of generations to run the algorithm for
c_fxCrossover function for the genetic algorithm, this function should re-populate the populations as well
r_fxPointer to a function that runs after every generation, with a pointer to the current ga_population and fitness score of the fittest candidate for the prior generation passed to it. NOTE: The next generation won't commence untile r_fx returns. This field can be set to NULL as well.
